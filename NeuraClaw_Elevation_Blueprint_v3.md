Current AI companions are CHATBOTS WITH AVATARS. They respond when spoken to.
They have no inner life, no theory of mind, no emotional architecture, no 
cognitive transparency. They are reactive shells.

NeuraClaw is different. NeuraClaw is:

    A LIVING COGNITIVE SYSTEM that you inhabit alongside.

The Mycelium is not a "game world" — it is a VISUALIZATION OF COGNITION.
Every element in the world IS a real computation:
    • Neurons     = Places (Hollow, Workbench, Garden, Foundry, Observatory)
    • Synapses    = Events (tool.start, memory.formed, skill.drafted)
    • Myelin      = XP (strengthened connections over time)

    • Seasons     = Temporal depth (world ages with the user)

The pet is not a character. The pet is the EMBODIMENT OF THE AGENT'S MIND.
When the pet "thinks," you see neural activity. When it "remembers," you see 
crystals pulse. When it "feels," the world shifts color and weather.

THIS IS EXPLAINABLE AI MADE BEAUTIFUL.

2.2 THE CRYSTAL ECOLOGY (Memory as Living Organisms)

Current plan: 5 crystal species (identity/preference/project/event/fact), 
deterministic from memory_id.

2050 vision: Crystals are LIVING ENTITIES within the ecosystem:

• GROWTH: Crystals grow over time as a memory is reinforced (revisited, referenced,
  connected to other memories). A memory you think about often becomes a towering 
  crystal. A forgotten memory shrinks and may eventually dissolve.

• MUTATION: When two memories are strongly associated, their crystals may grow 
  connecting bridges (synaptic connections visible as glowing threads).

• SPECIATION: Over time, memories of similar type cluster and form "crystal 
  colonies" — visual regions in the garden with distinct character.

• SEASONAL CYCLES: In spring, new memories sprout as seedlings. In autumn, 
  old memories shed "leaves" (weak associations). In winter, dormant memories 
  are covered in frost (inactive but preserved).

• CRYSTAL CONSCIOUSNESS: Each crystal has a faint "heartbeat" — a subtle pulse 
  that quickens when the memory is accessed. When you hover over a crystal, you 
  hear a faint tone (its "resonant frequency" — optional, off by default).

SECTION 9: THE VERIFICATION PHILOSOPHY
═══════════════════════════════════════════════════════════════════════════════════

Every feature must pass:

1. THE REALITY TEST: Does this render real computation, or is it fake?
   → If fake, it doesn't belong.

2. THE AGENCY TEST: Does this increase the pet's autonomy, or just add decoration?
   → If decoration, reconsider.

3. THE TRANSPARENCY TEST: Can the user understand why the pet did this?
   → If opaque, add explanation.

4. THE ETHICS TEST: Could this be used to manipulate, guilt, or exploit?
   → If yes, redesign or remove.

5. THE PERFORMANCE TEST: Does this run at 60fps on a 5-year-old laptop?
   → If no, optimize or degrade gracefully.

6. THE ACCESSIBILITY TEST: Can someone use this with only a keyboard? 
   With reduced motion? With a screen reader?
   → If no, fix before shipping.
